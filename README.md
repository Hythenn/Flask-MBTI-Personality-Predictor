## Project Overview

Linguistic analysis tool developed as a Lab Exercise to categorize text into Myers-Briggs personality types.
System architecture integrates Scikit-learn pipelines with a Python web interface for real-time inference.
Application provides descriptive analytics and visualization for the identified personality traits.

## Core Features

- Ensemble Classification: Utilizes multi-technique modeling for accurate personality type prediction.
- Data Visualization: Displays graphical representations of dataset distribution and analysis results.
- Web Integration: Features a Flask-based interface for user input and result presentation.
